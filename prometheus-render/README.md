# Prometheus for Render Deployment

This directory contains Prometheus configuration for monitoring the live HyperNova backend.

## Files:
- `prometheus.yml` - Prometheus configuration (scrapes live backend)
- `Dockerfile` - Docker image for Render deployment

## Backend URL:
- https://hypernova-backend-azgf.onrender.com/metrics

## Deployment:
Deploy this as a Docker service on Render.
